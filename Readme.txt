To launch the server:
- if you are on windows click and run the StartupWindows.bat script
- if you are on mac click and run the StartupMac.sh script.

if neither of these worked or if you are on linux open up your command 
line terminal and navigate to the Nodejs folder, then enter the command 
"node Core.js" to start the server. 

To connect to the website:
open your browser and connect to "localhost:3000"
